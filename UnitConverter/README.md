# Unit Converter

This project implements a versatile unit converter application that allows users to convert various units between one another. It supports conversions for temperature, length, area, and volume measurements. The user interacts with the program by entering the type of unit they are converting from, the type of unit they want to convert to, and the corresponding value. The program then performs the conversion and displays the result.

## Features

- **Multiple Conversion Categories**: The unit converter supports multiple categories, including:
  - Temperature: Celsius (°C), Fahrenheit (°F), Kelvin (K)
  - Length: Meter (m), Kilometer (km), Centimeter (cm)
  - Volume: Liter (L), Milliliter (mL), Cubic Meter (m³)
  - Area: Square Meter (m²), Square Kilometer (km²), Square Centimeter (cm²)

- **User-friendly Interface**: The program provides an intuitive graphical user interface (GUI) that allows users to input their conversion details easily.

- **Conversion Accuracy**: The conversion calculations are accurate, ensuring reliable results for users.
